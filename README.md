## ゲームストーリー  
ロボットに支配されてしまったこの世界。
荒れ果てた荒野にあなたは1人。
迫りくるロボットを撃ち抜き生き残れ！

## ゲームルール  
３つの武器を使って迫りくるロボットを撃ちまくれ。
どの武器にもオーバーヒートゲージがあるのでうまく交換しながら攻略しよう。
できるだけ多くのウェーブを生き残ろう。
